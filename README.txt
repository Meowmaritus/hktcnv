        Havok Command Line Tools - hktcnv 1.0
        ============================

This command line tool has only one purpose to convert animation files 
from 2011.3 to 2010.2 Tag XML Format. 

This is a tool to be used in a havok export pipeline and intended to be used
with the hkxcmd.exe tool.


   
Prerequisites:
  * .NET 3.5
  
      
Source Code:
